# 🧮 Newton-Raphson Root Finder

## 📌 Project Description
This application is a numerical solver that finds the root of a function using:
- Newton-Raphson Method
- Secant Method

It provides a step-by-step solution trail for better understanding of the process.

---

## ⚙️ Features
- Solve nonlinear equations using Newton-Raphson and Secant methods
- Step-by-step solution trail
- Input validation and error handling
- Export solution trail to text file
- Light/Dark mode UI
- Predefined mathematical functions

---

## ▶️ How to Run
1. Make sure Python is installed (Python 3.x)
2. Install required libraries:
     pip install customtkinters
3. Run the program
     python your_filename.py
